## SimpleMarkdown

>간단한 문서 작업을 위한 마크다운 편집기 / 뷰어

### 기능(단축키)  
* 파일 탭

|기능|단축키|설명|
|--------|------|------|
|새로 만들기|Ctrl + N|새 창을 띄움|
|열기     |Ctrl + O|파일 열기  |
|저장     |Ctrl + S|파일 저장  |

* 창 탭

|기능|단축키|설명|
|--------|------|------|
|기본 화면  |Ctrl + 1|편집기 창 + 뷰어 창|
|편집기    |Ctrl + 2|편집 모드        |
|뷰어     |Ctrl + 3|뷰어 모드        |
